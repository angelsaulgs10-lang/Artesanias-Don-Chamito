📱 1. ¿Qué es el diseño web responsivo?

El diseño web responsivo (RWD) permite que un sitio se adapte automáticamente al tamaño de pantalla: móviles, tablets, laptops y escritorio.

🎯 Objetivo: Experiencia óptima sin zoom ni desplazamiento horizontal.
📱 Mobile First: Diseñar primero para móviles, luego pantallas grandes.
🔍 Viewport: Controla cómo se muestra la página en dispositivos.
📐 Media Queries: CSS condicional según características del dispositivo.
📏 Unidades flexibles: %, rem, em, vw, vh en lugar de píxeles fijos.

👁️ 2. Meta Viewport: etiqueta esencial

La etiqueta <meta name="viewport"> es obligatoria para el diseño responsivo.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
AtributoValorSignificado
widthdevice-widthAncho igual al del dispositivo
initial-scale1.0Zoom inicial (1 = sin zoom)

📏 3. Unidades relativas

% (porcentaje)

Relativo al contenedor padre
width: 80%;

vw / vh

1% del ancho/alto de la ventana
font-size: 5vw;

rem

Relativo al tamaño de fuente del root
font-size: 1.2rem;

em

Relativo al elemento padre
padding: 2em;

/* Unidades relativas para diseño responsivo */ html { font-size: 16px; /* Base para rem */ } .contenedor { width: 90%; max-width: 1200px; margin: 0 auto; } .titulo { font-size: 2rem; } .imagen-responsive { max-width: 100%; height: auto; }

📐 4. Media Queries (breakpoints)

🎯 Breakpoints comunes: Móvil: hasta 600px | Tablet: 601px - 900px | Escritorio: más de 901px
/* Estilos base para móviles (Mobile First) */ .contenedor { display: flex; flex-direction: column; gap: 1rem; } /* Tablet: pantallas mayores a 768px */ @media screen and (min-width: 768px) { .contenedor { flex-direction: row; flex-wrap: wrap; } .tarjeta { width: calc(50% - 1rem); } } /* Escritorio: pantallas mayores a 1024px */ @media screen and (min-width: 1024px) { .tarjeta { width: calc(33.33% - 1rem); } }

💪 5. Flexbox + CSS Grid (herramientas clave)

Flexbox (1D): Ideal para menús, tarjetas, alineaciones simples.
CSS Grid (2D): Ideal para layout completo de la página.
/* FLEXBOX */ .contenedor-flex { display: flex; flex-wrap: wrap; justify-content: center; gap: 1rem; } .item-flex { flex: 1 1 250px; } /* CSS GRID */ .layout-principal { display: grid; grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)); gap: 1.5rem; } /* Grid con áreas (estructura completa) */ .pagina { display: grid; grid-template-areas: "header" "nav" "main" "aside" "footer"; } @media (min-width: 768px) { .pagina { grid-template-areas: "header header" "nav main" "aside aside" "footer footer"; grid-template-columns: 200px 1fr; } }
